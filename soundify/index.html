Soundify

For å starte soundify må du først starte backenden, den tillater CORS spørringer fra:

  • http://localhost:5173
  • https://localhost:5173
  • http://localhost:80
  • https://localhost:80
  • http://localhost:443
  • https://localhost:443
  • http://localhost:5500
  • https://localhost:5500

Da får vi håpe at du kjører fra en av disse. Om ikke kan du legge til addressen i "index.ts" filen.

Oppstart av backend

  1. Åpne et powershell eller bash vindu
  2. Gå til /soundify/backend/ og kjør "npm install && npm start"
  3. Det er det samme som å kjøre "node dist/index.ts" som er entrien for å starte api'en

Oppstart av frontend

Jeg klarte ikke å laste inn bilder som relative lenker, fordi miljøet for utvikling og produksjon er ulik.
Siden jeg ikke vet hvordan produksjons miljøet (din xampp) ser ut, må du starte frontenden i en egen server.

  1. Åpne et powershell eller bash vindu
  2. Gå til soundify/app/ og kjør "npm install && npm build && npm start"
